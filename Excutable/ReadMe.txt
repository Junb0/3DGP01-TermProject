2인용 로컬 대전 슈팅게임 입니다.

하나의 키보드로 두명이서 각각 자신의 탱크를 조작하여
상대방을 처치하면 됩니다.

원하는 해상도가 적힌 실행파일을 실행하면 됩니다.
다이렉트x12 및 비주얼 c++ 패키지가 설치되어 있어야 합니다.
실행파일과 같은 폴더 내에 Model 폴더와 Light.hlsl, Shaders.hlsl 파일이 존재해야 합니다.

플레이어중 한명이 사망하면 라운드가 종료되며 살아남은 플레이어의
점수가 오르고 잠시후 다음 라운드가 시작됩니다.

라운드가 지날수록 양측 탱크의 공격력이 증가합니다.
탱크의 시작 위치는 매 라운드마다 번갈아 가며 바뀝니다.

3점을 획득한 플레이어가 나타나면 게임이 종료됩니다.

아이템 효과 (해당 라운드만 적용)
사과 : 체력 50 추가
포도 : 최대 이동속도 25% 상승
체리 : 공격속도 20% 상승

조작법

플레이어 1 (초록)
W : 전진
S : 후진
A : 좌회전
D : 우회전
스페이스바 : 총알 발사

플레이어 2 (빨강)
방향키 상 : 전진
방향키 하 : 후진
방향키 좌 : 좌회전
방향키 우 : 우회전
엔터키 : 총알 발사

F9 : 전체화면
ESC : 게임 종료
